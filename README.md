# syspend

syspend recursively searches the parent directory and executes sys.path.append on the path where the SYSPEND_ROOT file exists.



project/
    ├── packages/
    │   ├── __init__.py
    │   ├── package1.py
    │   └── package2.py
    └── samples/
        └── sample1.py


## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install syspend
```
